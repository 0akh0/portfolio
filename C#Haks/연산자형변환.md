# 연산자와 형변환

## 산술 연산자

| 연산자 | 설명    |
|--------|---------|
| +      | 더하기  |
| -      | 빼기    |
| *      | 곱하기  |
| /      | 나누기  |
| %      | 나머지  |

## 비교 연산자

| 연산자 | 설명     |
|--------|----------|
| ==     | 같다     |
| !=     | 다르다   |
| >      | 크다     |
| <      | 작다     |
| >=     | 크거나 같다 |
| <=     | 작거나 같다 |

## 논리 연산자

| 연산자 | 설명  |
|--------|-------|
| &&     | AND  |
| \|\|   | OR   |
| !      | NOT  |

## 비트 연산자

| 연산자 | 설명      |
|--------|-----------|
| &      | 비트 AND  |
| \|     | 비트 OR   |
| ^      | 비트 XOR  |
| ~      | 비트 NOT  |
| <<     | 왼쪽 시프트 |
| >>     | 오른쪽 시프트 |

## 할당 연산자

- =, +=, -=, *=, /=, %= 등

## 조건(삼항) 연산자

```

string result = (score >= 60) ? "합격" : "불합격";

```

## 형변환

### 캐스트 연산자

```

int a = 10;
float b = (float)a;

```

### 메서드 이용

- ToString(), Parse(), Convert.~ 등

```

string s = 123.ToString();
int n = int.Parse("123");
double d = Convert.ToDouble("3.14");

```
